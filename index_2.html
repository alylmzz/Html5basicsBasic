Forms

Forms are needed for collecting user data given by user. Created with <form> tag <form action="">/<form> . The <action=""> attribute here means when we submit form which page it should lead us. Or we can link it to our JavaScript form, JSON form etc. which normally how its done. Normally only forms with a name attribute will be submitted. We must put name="" attribute to our labels and inputs which we will learn now. Example;
<form action="/07_forms_input.html" target="_blank"></<form>
Every form needes 2 basic tags, <label for="" id></<label> Which is the headline and data type for form and <input type="text" id> the data we collect. Colecting the data itself is about back-end development. Notice the id attributes in <label> and <input> tags. These are here to simply identfy them. so later we can call and use them in JS or CSS or remember in the label we created for="" attribute, it will link them also. now these two are link with "user", This means when someone clicks "User Name" in web page it will automatically highlight input and make it writable.

Types Of Forms

There are multiple types of forms like getting, e-mail, user name, passwords Etc. Here in the gif you see some of them and what it means "it will automatically highlight input and make it writable." All of the forms on this giff will be written and explained here as codes.

Text Input Form

Normal text input are small txt add area like names or user id's We can limit them to certain max charater limit using maxlength="" attribute. EXAMPLE maxlength="20" will be limited to 20 characters. We can also limit minimum charater limit with minlength="" EXAMPLE minlength="6" will ask user to write at least 6 characters. We can add a placeholder to text areas if we want to give message to user using placeholder="" EXAMPLE placeholder="Pasword must be at least 6 characters" . This will add a half seen message in input area which will dissapear as user starts typing If we want a text input to be a must be filled area we can add "required" attribute , this way page won't be submitted until that required input is filled like in passwords. For any inputs that are not required to be filled, we can leave an empty option And if we want we can use maxlength="", minlength="", placeholder="", required in any form input that uses text like textarea input (which we will learn shortly)
