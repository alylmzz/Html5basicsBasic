What is HTML?

HTML stands for "Hyper Text Markup Language" and it is the standard programming language for Web pages. Every web page has HTML elements are the building blocks of those pages and they are represented by elements called <> tags.

HTML Tags

Every Web Page has content inside to show the user what it consits of. And to understand what those things are and how to show them, it needs HTML Tags. HTML Tags are the hidden keywords within a web page that define how your web browser must format and display the content.

Creating A Tag

Most tags must have two parts, an opening and a closing part. For example, <html> is the opening tag and </html> is the closing tag. Closing tag has the same text as the opening tag, but has an additional forward-slash ( / ) character. But some tags are an exception , and where a closing tag is not required. The <img> tag for showing images is one example of this. Each HTML file must have the essential tags for it to be valid, so that web browsers can understand it and display it correctly.

Creating Your Tags

Very first thing we must put on an html file is the doctype. With this we tell the browser to read in spesific document type like HTML5 or XHTML or HTML4.0.1. Then what we need is an html tag looks like <html></html>. This tag defines the document as web page and identifies the beginning and end of the HTML document. We write all other tags between the html tags (<html>Right Here</html>). And inside it we need <head> and <body> tags. Rather then some exceptions like title, whatever is on <head> tag wont be seen by the user. It is for the developer to create spesifics of the site like style or meta tags which we will learn all about shortly. Inside head tag must be the <title> tag. Whatever name we give to the <title> tag will be shown name of the page on browsers tab. And final element of the <html> tag is the <body> tag. It simply consists of what we see on any site when loaded on internet, everyting we see as a user goes here.

Tag Attributes

Attributes are elements allow you to customise a tag. They are defined within the opening tag. For example,
<body id="home">.
Attributes are often assigned a value using the equals sign, such as class="" or id="", but there are some that only need to be declared in the tag like this: <hr noshade>. Most attributes are optional for most tags, and are only used when you want to change something about the default way a tag is displayed by the browser. However, some tags such as the <img> tag has required attributes such as src="" which is the source of the image. It is needed for the browser to display the web page properly. There are so many attributes to account for, for more information please visit w3schools

Meta Tags & Search Engines

Meta tags are tags or snippets that describes our pages contents. They don't appear in the body tag so nobody sees them. They just describe things about pages content. For example
<meta charset="UTF-8"> (charset)
is a meta tag specifies the character encoding of the page. Meta tags are also important for availabity to be found for a web page. Because search engines search through them to see what the page consist of. For example;
<meta name="description" content="This is Meta Tag example">.
Whatever we write instead of "This is Meta Tag example", will be seen as description on search engines. And adding keywords meta tag will enable search engines to look through them and to check if those words matches users' search. So adding spesific keywords about our page will make it easy to be found.

Links

When adding sources to our web page, we use links. There is an <a> tag and a <link> tag. For outside sources <link> tag an inside sources <a> tag are better options. For example <a href="http://google.com">Click For Google</a> will still take you to google even if anchor tag is build to link inside sources. href="" attribute is used for defining the source adress like in the google example. If an element, like another HTML page of our work, is an inside source we use forward-slash ( / ) character to search our computer. When linking outside pages to our website, a link will open it in the page you are in, to prevent that we must use target="_blank" attribute like
<a href="http://google.com" target="_blank"> Click For Google </a>
Seriously click it.

Images & Videos

Images & Videos <img src="" alt=""> this is the tag usage for images. src="" means source for image and alt="" means alternative text, meaning if image wont be shown for any reason, what text should be shown instead. Images can be linked from both inside and outside source. Same as links.
<img src="/images/sample.jpg" alt="My Image">
is a like to inside source. When we want to add a video to our page we use
<video src=""></video> tag.
But this alone will add the video as an image, to able to play, full screen etc. we need controls attribute in tag. Example ; <video controls src=""></video> . If we want the video to autoplay when the page loads we add autoplay attribute to tag ;
<video autoplay controls src=""></video>

Lists

There are 2 types of lists. Unordered Lists (by default gives items simple dots by order) written with <ul> tag, and Ordered Lists, (by default gives items numbers from 1 to end with order) written with <ol> tag. When we write items inside the lists, we use <li> tag. Ordered lits . Since lists tend to easily get styled within CSS, people tend to use unordered lists and style them as they like. Futhermore these lists are mainly used for navigation or side bar menu styling.

Nested Lists

To simply put it, list within list. Like book sections and chapters. Great use for multi layered menu styles. So take your time to learn these with unordered lists.
